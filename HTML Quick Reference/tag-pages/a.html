<a>

The <a> tag creates a hyperlink. You specify the URL of your link within the href element and the text of the hyperlink between the two tags. It can either be a link to an external website using the full URL of the website or a link to the a different page of the current website using a relative link.

Syntax

<a href="https://www.wikipedia.org/">Link to wikipedia<a>    <!-- External URL link -->

<a href="p.html">Link to <p> tag on this website<a>    <!-- Relative URL link -->
